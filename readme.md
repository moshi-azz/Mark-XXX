## ✨ Overview

**MARK XXXV** is an advanced voice-driven AI assistant designed to turn your computer into an interactive intelligent system.

Speak naturally — it listens, understands context, responds with a human-like voice, and executes tasks across your system automatically.

This version merges **Mark-XXX** (gesture control, agent system, flight finder) with **Mark-XXXV** (game updater, enhanced memory, F4 mute shortcut).

---

## 🚀 Capabilities

* **Real-time voice interaction** — Natural conversation with instant response
* **System control** — Launch apps, manage files, execute commands
* **Autonomous task execution** — Plans and completes multi-step workflows
* **Visual awareness** — Screen analysis and webcam understanding
* **Persistent memory** — Learns preferences, projects, wishes, and relationships
* **Hand gesture control** — Control the PC with your hands via webcam (MediaPipe)
* **Game management** — Update/install Steam & Epic Games automatically
* **Integrated tools** — Web search, weather, reminders, messaging, code help, flights
* **Desktop shortcut** — Arc Reactor icon launcher (`create_shortcut.py`)

## ⚡ Quick Start

```bash
git clone https://github.com/moshi-azz/Mark-XXX
cd Mark-XXX
pip install -r requirements.txt
playwright install chromium
python main.py
```

Enter your free Gemini API key on first launch.
System ready in minutes.

### Desktop Shortcut (optional)

Run once to create a JARVIS shortcut with the Arc Reactor icon on your Desktop:

```bash
python create_shortcut.py
```

## 📋 Requirements

* Windows 10 / 11
* Python 3.10 or newer
* Microphone
* Gemini API key (free at [aistudio.google.com](https://aistudio.google.com))

Key packages: `google-genai`, `pyaudio`, `pyautogui`, `opencv-python`, `playwright`, `mediapipe`, `pywinauto`

## 🤟 Gesture Control

| Gesture | Action |
|---|---|
| ☝️ Index finger | Move cursor |
| 🤌 Pinch (thumb+index) | Left click |
| 🤙 Thumb+middle | Right click |
| ✌️ Two fingers up | Scroll |
| 🖐️ Open palm | Screenshot |
| 👍 Thumb up | Toggle mute |
| 🤟 Three fingers | Next track |
| 🤙 Pinky up | Play / Pause |
| ✊ Fist (2 sec) | Stop gesture control |

## 🧠 Memory System

JARVIS remembers you across sessions with 6 categories:

- **Identity** — name, age, city, job, language
- **Preferences** — favorite food, music, games, hobbies
- **Projects** — ongoing work and goals you're building
- **Relationships** — friends, family, colleagues
- **Wishes** — future plans, things to buy, travel dreams
- **Notes** — habits, schedule, anything else
